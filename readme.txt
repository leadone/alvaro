у стрелок должен обязательные атрибуты name="left" и name="right"
у элемента который будет крутиться должны быть след. атрибуты:
	обязательные: name="scrollBox" *
	не обязательные: speed="x", где x - скорость прокрутки в милисекундах,
					 size_scroll="x", где x - ширина или высота в px одного блока из списка прокрутки,
					 count_scroll="x", где x - по сколько блоков крутить,
					 count_visible="x", где x - количество видимых блоков,
					 type_scroll="x", где x - равно либо top либо left(top для вертикальной прокрутки, left для горизонтальной)
	* если прокрутка вертикальная то атрибут type_scroll становиться обязательным
** рекомендуется для блока, который будет крутиться задавать ширину
*** если крутилка прокручивает блоки не корректно необходимо задать count_visible, count_visible и size_scroll
